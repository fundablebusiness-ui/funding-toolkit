## 🚀 Deployment

### GitHub Pages
1. Create a GitHub repository named `shesfundable.github.io`
2. Upload these HTML files
3. Pages will be live at: `https://shesfundable.github.io/`
   - Sales: `https://shesfundable.github.io/index.html`
   - Order: `https://shesfundable.github.io/order.html`
   - Thank You: `https://shesfundable.github.io/thank-you.html`

### Custom Domain
1. Add CNAME file with your domain
2. Update DNS settings with GitHub Pages IP addresses
3. Enable HTTPS in repository settings

### Systeme.io
1. Edit your Systeme.io page
2. Switch to HTML/Code editor
3. Copy-paste entire HTML code
4. Update button links to your actual checkout URL

## 🔧 Customization

### Update Links
Replace `#` with actual URLs:
- Newsletter signup: `Join the ShesNewsy Newsletter for exclusive funding tips and strategies.`
- Download buttons: Point to actual toolkit files
- Legal pages: Privacy, Terms, Refund Policy, etc.

### Payment Integration
Current form includes:
- Credit card fields (name, number, expiry, CVV, billing address)
- PayPal option toggle
- Form validation (client-side)

**To process payments:**
1. Add backend API connection
2. Integrate Stripe, PayPal, or Systeme.io payment processor
3. Handle payment processing and email delivery

### Email Configuration
Update these in your email system:
- Confirmation emails for orders
- Toolkit download links
- Welcome to ShesNewsy newsletter
- Support responses
